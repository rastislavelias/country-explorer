# Country Explorer

A simple web application for browsing countries, built with **React**, **TypeScript**, and **Vite**. It fetches country data from the [REST Countries API](https://restcountries.com/) and lets you search by name and filter by region.

## Key Features

- Browse countries with name, population, region, capital, and flag
- Search by country name (instant filter as you type)
- Filter by region (Africa, Americas, Asia, Europe, Oceania, etc.)
- Responsive layout with country cards
- Data loaded from REST Countries API

## Tech Stack

- **React** 19
- **TypeScript**
- **Vite** 7

## Running Locally

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

Build for production:

```bash
npm run build
```
